# diagrams

Exported architecture, sequence, ER, and flow diagrams.

## Source vs export
- Source files: `.mmd` (Mermaid), `.eraser`, or Figma links — editable
- Exports: `.svg` / `.png` — for docs and ADRs

## When to add one
A system or flow is being explained more than twice in prose. A diagram pays back the moment the third person asks.

## Naming
`diagrams/<topic>-<kind>.{mmd,svg,png}` — e.g., `diagrams/auth-sequence.mmd`.
