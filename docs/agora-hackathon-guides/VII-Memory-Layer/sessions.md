# sessions

Dated Claude session logs.

## What's in each note
- Header: date, branch, working dir, contracts in flight
- Timeline: tool events and key decisions (from log-to-vault hook)
- Footer: contracts closed, learnings, follow-ups (from stop-vault-log hook)

## Why
Reconstructable history of every working session. Future you can rebuild context without re-deriving it.

## Naming
`sessions/YYYY-MM-DD-<short-slug>.md`
