# Ship

Fifth and final stage. Land the work and close the loop.

## Purpose
Merge, deploy, and capture learnings so the next cycle starts smarter.

## Inputs
- Approved review
- Green CI

## Outputs
- Merged branch
- Deployed artifact (where applicable)
- Vault entry: what shipped, what was learned, follow-ups
- Closed contract

## Exit Criteria
Code is in main. Learnings are recorded in the Memory Layer. Any deferred work is filed as a new triage item — never as a vague "we'll get to it."
