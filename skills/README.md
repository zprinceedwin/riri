# Guides

Structured reference for the seven-layer workflow stack.

## Layers

1. **[I. Workflow](I-Workflow/)** — the five-stage pipeline: triage → contract → dispatch → review → ship
2. **[II. Completion Mandate](II-Completion-Mandate/)** — non-negotiables before any task is called done
3. **[III. Agent Layer](III-Agent-Layer/)** — role-based agents that own slices of work
4. **[IV. Skill Layer](IV-Skill-Layer/)** — invocable skills grouped by domain
5. **[V. Hook Layer](V-Hook-Layer/)** — automated harness behavior on session/tool events
6. **[VI. Config Layer](VI-Config-Layer/)** — settings, personas, commands, and protocols
7. **[VII. Memory Layer](VII-Memory-Layer/)** — persistence: vault, sessions, learnings, ADRs
