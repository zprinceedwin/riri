# Vault Learning

Mandatory capture: every shipped item leaves a trace.

## Rule
Before closing the contract, write a short vault entry. If nothing was surprising, write one sentence saying so.

## What to record
- What shipped (one line)
- Anything unexpected (a gotcha, a surprising failure, a stale assumption)
- Decisions worth replaying (why this approach over the alternative)
- Follow-ups filed as new triage items

## Where
Memory Layer → `learnings/` with date and a short title. Cross-link to the ADR if one was written.

## Why
Future sessions inherit context they would otherwise have to rediscover.
